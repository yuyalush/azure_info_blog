今週のハイライト
- Microsoft Discoveryが一般提供開始、エージェントAIワークフローの構築と管理が可能に。
- Azure Cobalt 200 VMsが発表、現代のエージェントAIワークロードに最適化された性能向上を実現。
- Microsoft Build 2026で発表された新しいアプリ開発の進展、Microsoft FabricとMicrosoft Databasesの統合。

## AI・機械学習
- **AI alone won’t change your business. The system running it will.**  
  AI単体ではビジネスを変えることはできません。重要なのは、それを支えるシステムです。Microsoftは、多様なモデルをサポートし、スタックの各層で柔軟性を提供する包括的なエージェントプラットフォームを構築しています。詳細は[こちら](https://blogs.microsoft.com/blog/2026/06/02/ai-alone-wont-change-your-business-the-system-running-it-will/)。

- **Microsoft Discovery一般提供開始とアプリプレビュー**  
  Microsoft Buildにおいて、Microsoft Discoveryが全組織向けに一般提供されることが発表されました。このプラットフォームは、エージェントAIワークフローの構築と管理を包括的にサポートします。詳細は[こちら](https://azure.microsoft.com/en-us/blog/announcing-microsoft-discovery-general-availability-and-microsoft-discovery-app-preview/)。

- **Foundry IQ: 統合された知識とサーバーレス検索で賢いエージェントを迅速に構築**  
  Microsoft Foundry IQを使用すると、企業データと外部データを統合し、安全でスケーラブルな知識レイヤーを構築できます。これにより、迅速かつ高品質な回答を実現します。詳細は[こちら](https://devblogs.microsoft.com/foundry/build-smarter-agents-faster-with-foundry-iq/)。

- **開発者向けガイド: Microsoft Foundryにおけるモデル、コスト、品質の管理**  
  Microsoft Foundryは、チームがAIをスケールで運用するための支援を行います。モデルの選定、評価、最適化、ガバナンスを通じて、AIのライフサイクル全体を管理します。詳細は[こちら](https://devblogs.microsoft.com/foundry/build-2026-foundry-models/)。

## コンピューティング
- **新しいAzure Cobalt 200 VMsが発表、50%の性能向上を実現**  
  Azure Cobalt 200 ARMベースの仮想マシンが発表され、LinuxベースのエージェントAIワークロードに最適化されています。これにより、性能が50%向上します。詳細は[こちら](https://azure.microsoft.com/en-us/blog/new-azure-cobalt-200-vms-deliver-50-performance-improvement-fully-optimized-for-modern-agentic-ai-workloads/)。

## 開発ツール
- **Microsoft Build 2026: Microsoft FabricとMicrosoft Databasesを用いたエージェントアプリの構築**  
  Microsoft Build 2026では、Microsoft FabricとMicrosoft Databasesを使用したアプリ開発の進展が強調され、スケーラブルでエージェントアプリケーションのための統合データおよびAIプラットフォームが紹介されました。詳細は[こちら](https://azure.microsoft.com/en-us/blog/microsoft-build-2026-building-agentic-apps-with-microsoft-fabric-and-microsoft-databases/)。
