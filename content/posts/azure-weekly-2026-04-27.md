今週のハイライト
- Azure Integrated HSMのオープンソース化により、信頼性と透明性が強化されました。
- MicrosoftがIDC MarketScapeのAPI管理部門でリーダーに選出されました。
- Azure Localにより、Microsoft Sovereign Private Cloudが数千のノードにスケール可能に。

## AI・機械学習
- **Microsoft Sovereign Private Cloud scales to thousands of nodes with Azure Local**  
  Azure LocalがMicrosoft Sovereign Private Cloudのスケーラビリティを向上させ、AIやデータワークロードを完全に制御し、コンプライアンスを確保しつつ、オフラインでの運用も可能にします。詳細は[こちら](https://blogs.microsoft.com/blog/2026/04/27/microsoft-sovereign-private-cloud-scales-to-thousands-of-nodes-with-azure-local/)をご覧ください。

## セキュリティ
- **Enforcing trust and transparency: Open-sourcing the Azure Integrated HSM**  
  Azure Integrated HSMがオープンソース化され、ハードウェアによる鍵保護がAzureに組み込まれました。これにより、シリコンからサービスまでの暗号的信頼が拡張され、透明性のある設計が実現されます。詳細は[こちら](https://azure.microsoft.com/en-us/blog/enforcing-trust-and-transparency-open-sourcing-the-azure-integrated-hsm/)をご覧ください。

## 開発ツール
- **Microsoft named a Leader in the IDC MarketScape: Worldwide API Management 2026 Vendor Assessment**  
  Azure API Managementが、従来のAPIからAIモデル、ツール、エージェントに至るまで、すべてを管理する単一のAzureネイティブプラットフォームを提供し、IDC MarketScapeでリーダーとして認められました。詳細は[こちら](https://azure.microsoft.com/en-us/blog/microsoft-named-a-leader-in-the-idc-marketscape-worldwide-api-management-2026-vendor-assessment/)をご覧ください。
